# DerbyLounge

Customer Management & POS system for gaming lounges. Built by [Derby Digital](https://derbydigital.us).

## Features
- **Customer Rolodex** — Track every customer with notes, tags, visit history, and total spend
- **Check-In System** — One-click check-in, live "Currently Here" view, minor flagging
- **Point of Sale** — Item catalog, cart, Stripe payments, digital receipts
- **Admin Dashboard** — Revenue charts, top customers, real-time stats
- **Inventory Management** — Stock tracking, low stock alerts, restock logs

## Tech Stack
- React + Tailwind + shadcn/ui
- Supabase (auth, database, real-time, edge functions)
- Stripe (payments)
